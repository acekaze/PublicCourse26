# 주력 강의 재설계 과정 피드백 기준표

## 기본 4축
1. 메시지가 선명한가
2. 구조가 자연스러운가
3. 참여가 설계되어 있는가
4. 전달과 전이가 가능하게 설계되어 있는가

## 세부 확인 질문

### 1. 메시지
- 이 강의가 왜 필요한지 분명한가?
- 학습자가 끝나고 무엇을 가져가야 하는지 보이는가?
- 핵심 메시지가 한 문장으로 정리되는가?

### 2. 구조
- 도입-전개-정리 흐름이 자연스러운가?
- 내용의 양과 순서가 적절한가?
- 중간에 학습자가 놓칠 지점이 없는가?

### 3. 참여
- 질문이 사고를 움직이게 설계되어 있는가?
- 활동이 단순 장식이 아니라 학습으로 이어지는가?
- 피드백이 수정 가능한 형태로 주어지는가?

### 4. 전달/전이
- 설명, 전환, 강조, 마무리 방식이 분명한가?
- 과정 후 실제 현장 적용으로 이어질 장치가 있는가?
- 실행계획이 구체적인가?

## 피드백 원칙
- 잘한 점 1개 이상 먼저 말한다.
- 보완점은 추상적 감상보다 수정 가능한 지점으로 말한다.
- 비판으로 끝내지 않고 다음 수정 방향까지 제안한다.
- 발표자 기준이 아니라 학습자 기준으로 본다.
