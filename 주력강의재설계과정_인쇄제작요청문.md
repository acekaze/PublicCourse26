## 2. 제작 사양 요청
- 사이즈: `A4`
- 제본: `무선제본`
- 표지: `컬러 인쇄`
- 내지: `흑백 인쇄`
- 표지 용지: `아트지 250g 내외`
- 표지 후가공: `무광 코팅`
- 내지 용지: `백색모조 100g` 또는 필기 가능한 유사 지종

## 3. 페이지 기준
- 표지: `1페이지`
- 내지: `32페이지`
- 합본 기준 전체: `33페이지`

## 4. 제작 시 유의사항
- 표지는 별도 파일 기준으로 사용 부탁드립니다.
- 내지는 워크북 성격이라 필기 공간이 중요합니다.
- 작성 박스와 메모 공간이 잘 보이도록 출력 농도와 선명도를 맞춰주시면 좋겠습니다.
- 내지 흑백 인쇄 시 선과 박스가 너무 연하지 않게 부탁드립니다.
- 첫 페이지 표지는 지금 제공된 파일을 그대로 사용하고, 뒷표지는 별도 디자인이 없으면 무지 처리해도 괜찮습니다.

```text
제본은 무선제본, 표지는 무광 코팅, 내지는 필기 가능한 용지로 부탁드립니다.
내지의 작성 박스와 메모 공간이 선명하게 보이도록 출력 농도 확인 부탁드립니다.
```
