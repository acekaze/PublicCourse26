# 주력 강의 재설계 과정 운영진 역할분담표 초안

## 문서 목적
- 강사 혼자 모든 것을 떠안지 않도록 역할을 분리하는 운영 문서다.
- 현장에서는 작은 누락이 전체 흐름을 흔들기 때문에, 누가 무엇을 책임지는지 미리 확정해야 한다.
- 운영 자산화 관점에서도 역할 기준이 있어야 다음 기수에 사람만 바꿔 재사용할 수 있다.

## 권장 역할 구조
### 1. 총괄 강사
- 과정 목표와 메시지 유지
- 핵심 설명 진행
- 주요 피드백 제공
- 시간 지연 시 우선순위 조정 결정
- 어려운 참가자 개입 판단

### 2. 운영 담당
- 출석, 좌석, 명찰, 교재, 식사, 링크, 제출물 관리
- 휴식 및 복귀 시간 공지
- 참가자 문의 1차 응대
- 결제/영수/현장 안내 확인
- 후기, 동의서, 사진 기록 회수

### 3. 기록 및 보조 담당
- 소그룹 기록 체계 점검
- 주요 발언, 막힌 지점, 우수 사례 후보 메모
- 사진 또는 현장 기록 수집
- 후속 사례화 가능한 포인트 표시
- 사후 아카이브 정리 초안 작성

## 인력 최소 기준
- 최소 2인 운영: 강사 1 + 운영 1
- 권장 3인 운영: 강사 1 + 운영 1 + 기록보조 1
- 참가자 10명 이상이면 기록보조 1인을 두는 것이 안전하다.

## 역할별 상세 체크리스트
### A. 총괄 강사 체크리스트
#### 사전
- [ ] 과정 목표와 2일 흐름 최종 확인
- [ ] 오프닝 멘트와 마무리 멘트 정리
- [ ] 공통 피드백 기준 재확인
- [ ] 예상 질문과 어려운 상황 대응 문장 준비
- [ ] 미니 시연 진행 기준 확정

#### 당일
- [ ] 오프닝에서 과정 목적과 규칙을 분명히 설명
- [ ] 설명 시간이 길어지지 않게 통제
- [ ] 참가자별 진행 속도 차이를 읽고 개입
- [ ] 피드백을 수정 가능한 문장으로 제공
- [ ] 마지막 세션에서 실행계획까지 반드시 완료

#### 사후
- [ ] 핵심 후기 검토
- [ ] 우수 사례 후보 선정
- [ ] 다음 기수 수정 포인트 3~5개 기록

### B. 운영 담당 체크리스트
#### 사전
- [ ] 참가자 명단 확정
- [ ] 결제 완료자 기준 좌석 확정
- [ ] 사전진단 회수 현황 업데이트
- [ ] 대표 강의 자료 제출 여부 확인
- [ ] 명찰, 교재, 안내문, 제출 링크 준비
- [ ] 식사/다과/공간/장비 최종 확인

#### 당일
- [ ] 출석 체크와 지각자 확인
- [ ] 교재 및 자료 배포
- [ ] 휴식 시간, 식사 시간, 종료 시간 공지
- [ ] 제출물 수집 현황 실시간 체크
- [ ] 문의 및 돌발 상황 1차 응대
- [ ] 후기 및 동의서 회수

#### 사후
- [ ] 참석/결석/지각 기록 정리
- [ ] 제출물 목록 정리
- [ ] 후기 원문 정리
- [ ] 사후 안내 발송 대상 확인

### C. 기록 및 보조 담당 체크리스트
#### 사전
- [ ] 기록 양식 준비
- [ ] 사진/메모 기준 정리
- [ ] 우수 사례 표시 기준 공유

#### 당일
- [ ] 세션별 막힌 지점 기록
- [ ] 좋은 피드백 문장 기록
- [ ] 참가자 반응 포인트 기록
- [ ] 우수 사례 후보 표시
- [ ] 사진 또는 장면 기록 확보

#### 사후
- [ ] 운영 회고용 메모 정리
- [ ] 사례화 가능한 원문 추출
- [ ] 다음 기수 재사용 항목 분류

## 참가자 문의 대응 기준
### 운영 담당이 바로 답할 수 있는 것
- 일정, 장소, 준비물
- 점심/다과/와이파이/주차 여부
- 제출 방식과 제출 마감
- 영수증 또는 기본 결제 확인

### 강사 판단이 필요한 것
- 과정 적합 여부
- 자료가 부족한 참가자의 참여 가능 판단
- 피드백 방식 관련 예외 요청
- 시연 범위 조정 요청

## 운영 직전 최종 확인 회의 안건
1. 현재 결제 완료 인원은 몇 명인가
2. 사전진단과 자료 제출 미완료자는 누구인가
3. 소그룹 편성은 끝났는가
4. 미니 시연 순서는 어떻게 잡을 것인가
5. 사진/후기/추천사 활용 동의는 어떻게 받을 것인가
6. 지각, 결석, 장비 이슈 발생 시 누가 먼저 움직이는가

## 이 문서가 특히 필요한 이유
- 지금 있는 운영체크리스트는 해야 할 일의 목록이다.
- 하지만 실제 현장에서는 목록보다 책임자가 더 중요하다.
- 역할이 분리되지 않으면 강사가 시간 관리, 문의 응대, 피드백, 자료 회수를 동시에 떠안게 된다.
- 이 문서가 있어야 운영 품질을 사람 의존에서 구조 의존으로 옮길 수 있다.
