# 교수법 공개과정 사전진단 폼 문항 초안

## 기본 정보
- 이름
- 연락처
- 이메일
- 소속 또는 활동 분야

## 현재 강의 정보
- 현재 운영 중인 대표 강의 제목은 무엇인가요?
- 이 강의는 주로 누구를 대상으로 하나요?
- 강의 시간은 보통 얼마나 되나요?
- 현재 가지고 있는 자료는 무엇인가요? (강의 개요 / 교안 / 슬라이드 / 소개문 / 기타)

## 현재 상태 진단
- 현재 대표 강의의 목적을 한 문장으로 적어주세요.
- 이 강의가 끝난 뒤 학습자에게 무엇이 남아야 하나요?
- 현재 강의에서 가장 아쉬운 지점 2~3가지를 적어주세요.
- 아래 항목 중 가장 먼저 고치고 싶은 것은 무엇인가요?
  - 메시지 선명도
  - 강의 구조
  - 질문 설계
  - 활동 설계
  - 피드백 방식
  - 전달 방식
  - 마무리 및 전이

## 과정 기대 결과
- 이번 과정에서 꼭 개선하고 싶은 것 1가지는 무엇인가요?
- 과정이 끝난 뒤 어떤 상태가 되면 "잘 왔다"고 느낄 것 같나요?
- 미니 시연과 피드백 참여에 동의하시나요?

## 제출 자료 관련
- 대표 강의 자료를 사전 제출할 수 있나요?
- 제출 가능한 자료 종류를 적어주세요.
- 사전 제출 예상 가능일을 적어주세요.
